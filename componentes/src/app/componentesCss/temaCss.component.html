CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para controlar la presentación y el diseño de una página web. Es uno de los tres pilares fundamentales del desarrollo web, junto con HTML y JavaScript. CSS funciona mediante la creación de reglas que se aplican a los elementos HTML de una página web. Estas reglas definen cómo se presentará el contenido de una página web, incluyendo cosas como el color, la fuente, el tamaño y la posición de los elementos en la página.
